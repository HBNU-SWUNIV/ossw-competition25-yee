### 기술 스택

### 설치 방법
```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

### 프로젝트 구조
```
frontend/
├── src/           # 소스 코드
├── public/        # 정적 파일
└── README.md
```
